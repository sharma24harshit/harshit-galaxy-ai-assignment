# 🚀 Project Name

## 📌 Description

This project is a modern web application built to handle user registration, authentication, and referral-based workflows. It includes features like adding nodes and then connecting and interacting with those nodes.

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React
* **Backend:** Node.js, Express.js
* **Database:** PostgreSQL (hosted on Neon)
* **ORM:** Prisma
* **Authentication:** OTP-based auth flow

---

## 🌐 Deployment

* **Frontend:** Vercel
* **Database Hosting:** Neon

---

## ✨ Key Features

* 🔐 Secure user authentication with OTP
* ⚡ Optimized database transactions using Prisma
* 📱 Responsive UI with modern design
* Node Creation

---

## ⚙️ Setup (Basic)

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```
